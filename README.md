# World Happiness Report Exploration

## Objective
Analyze and visualize factors affecting happiness scores across countries.

## Dataset
World Happiness Report Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Correlation Analysis
4. Country-wise Happiness Comparison
5. Visualization of Key Factors

## Key Insights
- Countries with higher GDP tend to have higher happiness scores.
- Social support strongly influences happiness.
- Regional differences can be observed in happiness rankings.

## Conclusion
The project explores the major factors contributing to happiness and provides visual insights using data analysis techniques.
